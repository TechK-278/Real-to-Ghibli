# Model Information

The project uses a **Diffusion Model converted to ONNX format** for faster inference.

## Model Format

ONNX

## Runtime

ONNX Runtime

## Advantages

* Faster inference
* Cross platform support
* Lower memory usage

## Pipeline

Input Image
↓
Preprocessing
↓
Diffusion Model (ONNX)
↓
Post Processing
↓
Generated Ghibli Image
