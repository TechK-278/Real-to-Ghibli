# 🎨 Ghibli Art Generator

A web application that converts images into **Studio Ghibli-style artwork** using a **Diffusion Model optimized with ONNX Runtime**.

The system contains a **Python backend for AI inference** and a **frontend interface for image upload and visualization**.

---

## 🚀 Features

* Generate **Ghibli-style art from images**
* Diffusion-based image transformation
* ONNX optimized inference
* Simple frontend interface
* Fast backend API

---

## 🧠 Tech Stack

* Python
* ONNX Runtime
* Diffusion Models
* HTML / CSS / JavaScript
* Flask / FastAPI
